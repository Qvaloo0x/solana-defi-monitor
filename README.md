# Solana DeFi Intelligence Monitor

Analytical framework to monitor Solana's economic health and DEX activity using Dune SQL.

## Dashboard Components

* **DEX Dominance**: 30-day volume distribution by protocol.
* **Network Velocity**: Daily transaction count (Swaps) to measure adoption.
* **Market Sentiment**: Top 10 assets by volume in the last 24 hours.

## Technical Details

* **Data Source**: `dex_solana.trades`
* **Network**: Solana Mainnet
* **Update Frequency**: Real-time (on-chain)
